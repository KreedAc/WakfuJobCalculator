# Wakfu Job Calculator

Unofficial toolkit for [WAKFU](https://www.wakfu.com/) players — live at [wakfujobcalculator.com](https://wakfujobcalculator.com/).

**Tools included:**

- **XP Calculator** — crafts needed to level each profession
- **Sublimations** — searchable database with slot/color filtering
- **Items Craft Guide** — recipe trees and shopping lists
- **Treasures** — treasure hunt achievements tracker
- **Combat Calculator** — damage, heals, armor, EHP, lock/dodge, resistances and more
- Guides and multi-language support (EN, FR, ES, PT)

WAKFU is an MMORPG published by Ankama. This is an unofficial fan site with no connection to Ankama.

## Tech stack

React 18 + TypeScript + Vite + Tailwind CSS. Fully static: all game data is served as JSON files from `public/data/` — no backend or database required.

## Development

```bash
npm install
npm run dev        # local dev server
npm run build      # production build → dist/
npm run typecheck  # TypeScript check
npm run lint       # ESLint
```

## Deploying (static hosting)

The site is a single-page app. Any static host works; it needs:

1. **Build command:** `npm run build`
2. **Output directory:** `dist`
3. **SPA fallback:** all paths must serve `index.html`. The included `public/_redirects` file (`/* /index.html 200`) handles this automatically on Netlify and Cloudflare Pages.

No environment variables are required.

### Cloudflare Pages (recommended)

1. Dashboard → **Workers & Pages → Create → Pages → Connect to Git** and select this repository.
2. Build command `npm run build`, build output directory `dist`.
3. After the first deploy, add the custom domain under **Custom domains**. If the domain's DNS is managed elsewhere (e.g. Aruba), the simplest path is moving DNS to Cloudflare by updating the nameservers at the registrar.

### Netlify

1. **Add new site → Import an existing project** and select this repository.
2. Build command `npm run build`, publish directory `dist`.
3. Add the custom domain under **Domain management** and follow the DNS instructions.

## Updating game data

The JSON datasets in `public/data/` are generated by the scripts in `scripts/`:

```bash
npm run fetch:treasures     # rebuild treasures data
npm run fetch:i18n:client   # extract i18n from the Wakfu client
npm run treasures:i18n      # build treasure translations
```
